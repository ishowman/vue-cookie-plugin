# vue-cookie-plugin

## 安装依赖
```
yarn add
```

### 本地调试
- 在当前目录运行 `yarn link`(或 `npm link`)
- 在本包的消费方，执行`yarn link "vue-cookie-plugin"`
- 在当前目录运行 `npm run dev`

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
